# Enduro Docker

## Installation

Use make:
```
usage: make install
       make uninstall
```

## Usage:
```shell
$ enduro --version # To confirm it is not locally installed
zsh: command not found: enduro
make install
...
$
$ enduro --version
1.4.16
$
```
